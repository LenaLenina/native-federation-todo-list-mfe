# Mfe

## Commands

npm install -g @angular/cli@16

ng version

ng new mfe --create-application=false

ng g application mfe --routing --style=scss

ng add @angular-architects/native-federation@16.2.0 --project mfe --port 4201 --type remote

ng serve mfe -o

ng build mfe
